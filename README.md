## Technologies Used

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, TypeScript, MongoDB, Socket.io, Firebase Admin SDK

## Deployed URL

The application is deployed at: [https://chat-app-51gt.vercel.app/home](https://chat-app-51gt.vercel.app/home)

## How to Run the Application Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/kyoka-miy/chat-app.git
   ```

2. Navigate to the project directory:

   ```bash
   cd chat-app
   ```

3. Install dependencies for both frontend and backend:

   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```

4. Start the backend server:

   ```bash
   cd backend
   npm run dev
   ```

5. Start the frontend development server:

   ```bash
   cd ../frontend
   npm run dev
   ```

6. Open your browser and navigate to:
   ```
   http://localhost:3001
   ```

Enjoy chatting!
